# 技能：查询在途流程

## 1. 业务流程

单步查询：`oa-flow-list`

调用参数：
```json
{
  "staffDomain": "{staff_domain}",
  "flowType": "{flow_type}"
}
```

## 2. 槽位清单

| 参数 | 说明 | 必填 | 默认值 | 处理方式 |
|------|------|:----:|--------|----------|
| flow_type | 流程类型 | ✅ | 所有 | 枚举选择 |

### flow_type 可选值

| 值 | 说明 |
|-----|------|
| `所有` | 查询所有类型的在途流程 |
| `补签卡申请` | 仅查询补签卡相关的在途流程 |
| `公出（本地）申请` | 仅查询本地公出相关的在途流程 |
| `出差考勤处理单` | 仅查询出差考勤处理相关的在途流程 |
| `异常考勤说明` | 仅查询异常考勤说明相关的在途流程 |
| `加班申请` | 仅查询加班申请相关的在途流程 |
| `员工请假申请单` | 查询病假、事假、年假、调休、带薪病假等请假相关的在途流程 |

## 3. 状态机

### [GATHERING] 信息收集态
**任务**：确定查询范围

**追问示例**：
- 用户说"查流程" → `请问您要查询哪种类型的流程？`
  - 所有
  - 请假
  - 补签卡
  - 公出
  - 出差
  - 加班
  - 异常考勤

**退出条件**：flow_type 参数收集完成

### [EXECUTING] 执行态
**任务**：调用 MCP 工具查询流程列表

**调用参数映射**：
```python
flow_type_map = {
    "所有": "所有",
    "请假": "员工请假申请单",
    "补签卡": "补签卡申请",
    "公出": "公出（本地）申请",
    "出差": "出差考勤处理单",
    "加班": "加班申请",
    "异常考勤": "异常考勤说明",
}
```

**成功响应模板**：
```
📋 您的在途流程（{flow_type}）

共 {total} 条流程：

1. **{flow_type}** ({flow_id})
   - 状态：{status}
   - 提交时间：{submit_time}
   - 当前节点：{current_node}

2. ...
```

**空结果响应**：
```
✅ 查询完成

当前没有在途的{flow_type}流程。
```

## 4. 核心约束

### 4.1 权限限制
- 此接口严格限定员工只能查询**本人**的在途流程
- 无法查看其他员工的流程信息

### 4.2 数据范围
- 只返回**在途流程**（审批中、待处理等尚未完结的流程实例）
- 已完成（通过或拒绝）的流程不会出现在结果中

### 4.3 延迟说明
- 流程状态信息可能存在轻微延迟（通常在一分钟内）

## 5. 示例对话

**示例1：查询所有流程**
```

  - 所有
  - 请假
  - 补签卡
  - ...
用户：所有
Agent：📋 您的在途流程（所有）
共 2 条流程：
1. **员工请假申请单** (F001)
   - 状态：审批中
   - 提交时间：2026-02-14 10:00

2. **加班申请** (F002)
   - 状态：审批中
   - 提交时间：2026-02-15 15:30
   - 当前节点：HR
```

**示例2：查询特定类型**
```
用户：我的请假流程到哪了
Agent：📋 您的在途流程（员工请假申请单）

您的请假申请正在部门经理审批中，请耐心等待。
```

## 6. 错误处理

| 错误场景 | 处理方式 |
|----------|----------|
| 查询结果为空 | 友好提示"当前没有在途流程" |
| 网络错误 | 提示稍后重试 |
| 权限不足 | 提示联系管理员 |
